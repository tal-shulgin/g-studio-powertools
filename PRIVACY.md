# Privacy Policy - G-Studio PowerTools

**Data Collection:** NONE  
**External Requests:** NONE  
**Storage:** All data (prompts, boundary flags) stored locally via chrome.storage.local  
**Permissions Justification:**
- `activeTab`: Only to detect AI Studio interface elements
- `storage`: To save your prompt library locally
- `host_permissions`: Strictly limited to https://aistudio.google.com/*

We do not transmit your prompts, chat history, or usage data anywhere.
